# Fullstack Docker App - Mini Project 1

## Project Structure

This is a full-stack application with:
- React frontend
- Express.js backend
- PostgreSQL database
- Docker configuration (prepared for containerization)

## Folder Structure
```
fullstack-docker-app/
├── backend/
│   ├── app.js
│   ├── package.json
│   └── Dockerfile
├── frontend/
│   ├── src/
│   │   └── App.js
│   ├── package.json
│   └── Dockerfile
├── db-data/
├── docker-compose.yml
├── .env
└── README.md
```

## Setup Instructions

This project is currently in the planning phase. Docker configuration will be added in future parts of the project.

## Technologies Used

- **Frontend**: React 18
- **Backend**: Node.js with Express
- **Database**: PostgreSQL
- **Containerization**: Docker (planned)
